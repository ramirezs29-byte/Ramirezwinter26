CSS Properties

CSS properties are used to style and format the content of a web page. They allow you to control the layout, colors, fonts, and other visual aspects of your website. Here are some common CSS properties:

color and background properties

Color

sets the text color for different elements.Specifies the color of text.

backround-color

sets the backround color behind the content of an element.

Background-image

sets an image as the backround of an element.Adds texture,color,depth,or context.

Font family

specifies which font (text style) should be used.Shares common design,but vary in weight,style,or width

Font size

sets up how large or small the text appears.

text allign

controls how the text is aligned like left,right,context.

line height

sets the amount of space between lines of text.Controls the vertical spacing between lines of the text.

Box model properties

Margin

creates space around elements, outside of any defined borders.

Margin Top

sets the top margin of an element.Adds spare above an element.

Margin Bottom

sets the bottom margin of an element.Adds space below an element.

Padding creates space around an element's content, inside of any defined borders.

border

sets the border around an element. It can be used to add a line or box around an element, and you can customize its width, style, and color.

Border

sets the border around an element. It can be used to add a line or box around an element, and you can customize its width, style, and color.

border radius

sets the rounded corners of an element. It allows you to create smooth, curved edges for boxes and other elements on your web page.

display

determines how an elements is shown on the page(block,relative,absolute,fixed,sticky)

position

controls how an element is positioned on the page. It can be used to specify whether an element should be positioned relative to its normal position, or if it should be fixed in place on the screen.

float

moves an element to the left or right side of the container,allowing text and allign
